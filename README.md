# STM32 BQ769x0 4-10 BMS（电池管理系统）

## 📋 项目简介

BMS（电池管理系统）是目前新能源（EV）、储能（PV/ESS）和机器人领域的绝对核心。本项目是一个基于STM32微控制器和BQ769x0电池保护芯片的电池管理系统实现，适合电子工程初学者学习和实践。

## 🎯 项目特点

- ✅ 基于STM32微控制器
- ✅ 集成BQ769x0电池保护芯片
- ✅ 支持4-10节锂电池管理
- ✅ 完整的I2C驱动实现
- ✅ 从基础到高级的递进式设计
- ✅ 适合初学者快速上手

## 📁 项目结构

```
├── README.md                  # 项目说明文档
├── 总体计划                    # 项目整体规划和架构设计
├── Level 1 I2C驱动            # I2C通信驱动实现
└── BQ芯片的处理               # BQ769x0芯片配置和处理逻辑
```

## 🔧 核心模块

### 1. **Level 1 I2C驱动**
   - STM32 I2C外设配置
   - 与BQ769x0芯片的通信接口
   - 数据读写操作

### 2. **BQ芯片的处理**
   - BQ769x0芯片的初始化配置
   - 电压、电流、温度等参数读取
   - 保护功能实现（过压、欠压、过流、过温等）

### 3. **总体计划**
   - 系统架构设计
   - 功能规划和开发路线图
